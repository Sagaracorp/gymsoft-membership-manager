WebcamPlayer - Readme
=====================

This example application shows how to capture images
from a webcam using WPF

For more information:
http://blog.catenalogic.com/post/2008/12/22/Retrieving-snapshots-from-a-webcam.aspx

Original code is from:
http://blogs.microsoft.co.il/blogs/tamir/archive/2008/04/23/webcam-control-with-wpf-or-how-to-create-high-framerate-player-with-directshow-by-using-interopbitmap-in-wpf-application.aspx

The J832 library is downloaded from:
http://j832.com/
